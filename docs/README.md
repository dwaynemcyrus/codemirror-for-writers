# Docs Index

`codemirror-for-writers` is a CodeMirror 6 extension library for hybrid markdown editing.

For the full API reference and feature list, use the root `README.md`.

This `docs/` folder contains focused guides:

- `docs/BUILD.md`: local development, build, publish, and deploy notes.
- `docs/INTEGRATION.md`: integration patterns for React/Next.js apps.
- `docs/upgrades.md`: feature roadmap/history notes.
